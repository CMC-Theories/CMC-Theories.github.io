Java Calendar

This was a team project to build a calendar application from start to finish, where we were given just the requirements for the application (like it must be able to store events), but was open ended from there.
This project was to learn about how to go about the entire software development cycle, from planning, to development, to testing and to release.
Our group decided to implement this project in java.

So what did I implement?

  • All of the graphics, using Java's JFrame classes.
  • The underlying logic for performing C.R.U.D like operations.
  • Saving and loading events in the user system.
  • Some documentation regarding classes and use cases.

As with all of my team projects, I contributed with a personal account.
