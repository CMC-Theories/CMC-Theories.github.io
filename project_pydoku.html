Pydoku

This was a programming challenge to build a program in python or a similar programming language in under 24 hours.

My time: 21 hours, 14 minutes!

Okay, I decided to get some sleep during this so in reality, it is more like only 11 hours.
This included everything from learning how to actually use the python package, to implementing a game with it!

So what did I implement?

  • Semi-arbitrary board size (1^2x1^2 -> 5^2x5^2)
  • Live feedback for violations
  • An automated solver using backtracking
  • Undo and Redo capabilities
