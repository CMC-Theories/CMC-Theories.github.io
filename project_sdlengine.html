Stylistic Agents

This was an exploritory project where I taught myself about how to use computation shaders and what they are good for. I wanted to explore how to use computational shaders, but I also wanted to explore creating simple agents, so I implemented "birds" (represented as wisps on screen).

Each of these "birds" will simply

  1. Go straight if nothing is in their way.
  2. Dodge if something is in their way.
  3. Fly away for birds that are way too close.
  4. Fly in the same direction as nearby birds.
  5. Fly towards the center of the their flock.

So what did I implement?

  • Graphically, some glowing wisps that fly around, dodge walls.
  • Agents will generate tones that are mixed at the scripting level based on their locations in space.
  • Heading behavior calculated with computation shaders.
  • Stress test to see if computation shader perform better or worse than just the CPU in this project.

For the last point, I did notice about a factor of 3x speed up when using the computation shaders, however it was running at about 2 seconds per frame. This was with about 2,000 agents, which means about 2 million comparisons are done per frame. For the music generation, it isn't terrible with only 3 agents in the world scene, but more than that becomes a cacophony. Check out the source code!
