The Final Lock

This was a team made unity game. We chose the theme of escape rooms with the intent on making it a multiplayer experience, sadly this didn't work out as well as we had hoped, as we were approaching this by using the low level networking API instead of the then high level api (which was being depreciated at the time).
There were five levels created, which included sound, objects to interact with, music, environmental effects, animations, etc.

So what did I implement?

  • Most of the logic for interacting with elements and chaining the effects.
  • Level and puzzle designs, implementing logic for most puzzles (or puzzle components).
  • Multiplayer/networking code.

As with all of my team projects, I contributed with a personal account.
